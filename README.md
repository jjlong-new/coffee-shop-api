# Coffee Shop API ☕

A Spring Boot backend application for managing coffee orders.

## 📌 Features
- Create and manage coffee orders
- External pricing service integration
- RESTful API design
- Global exception handling
- Unit testing

## 🛠 Tech Stack
- Java 17+
- Spring Boot
- Spring Data JPA
- WebClient
- Maven

## 🚀 Future Enhancements
- MongoDB logging (order_logs)
- Analytics dashboard
- Order history tracking

## ▶️ How to Run
1. Clone the repo
2. Run `mvn spring-boot:run`
3. Access API at `http://localhost:8080`
